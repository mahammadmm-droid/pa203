# 💻 PA203 Proqramlaşdırma Qrupu

## 📖 Layihə haqqında

**PA203** proqramlaşdırma sahəsində bilik və bacarıqlarını inkişaf etdirən tələbələrdən ibarət bir qrupdur. Bu repository Git və GitHub texnologiyalarını öyrənmək, versiya nəzarət sistemindən düzgün istifadə etmək və komanda şəklində işləmə bacarıqlarını inkişaf etdirmək məqsədilə yaradılmışdır.

# 🌿 Git Workflow

Layihə aşağıdakı Git mərhələləri üzrə hazırlanmışdır:

1. Yeni GitHub repository yaradıldı.
2. Repository lokal kompüterə klonlandı.
3. `Group.txt` faylı yaradıldı.
4. Fayl `main` branch-ə əlavə edildi.
5. `branch1`, `branch2` və `branch3` adlı üç fərqli branch yaradıldı.
6. Hər branch-də fərqli məzmuna malik fayllar əlavə edildi.
7. Dəyişikliklər commit edildi və GitHub-a göndərildi.
8. Bütün branch-lər `main` branch-i ilə merge edildi.

# 🛠 İstifadə olunan Git əmrləri

Repository-ni klonlama

```bash
git clone <repository_url>
```

Repository qovluğuna keçmək

```bash
cd PA203
```

Yeni branch yaratmaq

```bash
git checkout -b branch1
```

Dəyişiklikləri əlavə etmək

```bash
git add .
```

Commit yaratmaq

```bash
git commit -m "Yeni fayl əlavə edildi"
```

GitHub-a göndərmək

```bash
git push origin branch1
```

Main branch-ə keçmək

```bash
git checkout main
```

Merge etmək

```bash
git merge branch1
git merge branch2
git merge branch3
```

Son dəyişiklikləri GitHub-a göndərmək

```bash
git push origin main
```
